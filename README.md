# financenews
🏛️ Politico-Finance News & Beta DashboardA fully automated, serverless news aggregator that tracks the intersection of global politics and financial markets. This project uses GitHub Actions to scrape RSS feeds, perform sentiment analysis, and deploy a live dashboard to GitHub Pages—100% free.🚀 FeaturesAutomated Scraping: Uses Python and feedparser to pull real-time data from Reuters, CNBC, and Financial Times.Impact Analysis: Leverages the VADER sentiment analysis tool to determine if political news is Bullish, Bearish, or Neutral.Zero Cost: No paid API tokens (OpenAI/NewsAPI) required. It uses open RSS standards.Self-Updating: GitHub Actions refreshes the data every 6 hours and commits the latest insights directly to the repo.🛠️ Tech StackFrontend: HTML5, Tailwind CSS (via CDN)Backend: Python 3.xAutomation: GitHub ActionsData Format: JSON⚙️ Setup & InstallationTo get your own version of this dashboard running, follow these steps:1. Fork the RepositoryClick the Fork button at the top right of this page to create your own copy.2. Enable GitHub Actions PermissionsBecause the script needs to "write" the news data back to your repository, you must grant it permission:Go to your repo Settings > Actions > General.Scroll down to Workflow permissions.Select Read and write permissions.Click Save.3. Activate GitHub PagesGo to Settings > Pages.Under Build and deployment, set Source to Deploy from a branch.Select the main branch and the /(root) folder.Click Save.4. Run the First UpdateGo to the Actions tab.Select the Update Financial News workflow on the left.Click the Run workflow dropdown and hit the button.Once finished, your site will be live at https://yourusername.github.io/your-repo-name/.📊 How the "Betting" Logic WorksThe system evaluates articles based on a scoring algorithm ($S$):$$S = \text{Sentiment Score} \times \text{Keyword Weight}$$Political Keywords: Election, Tariff, Sanction, Bill, Regulation.Financial Keywords: Fed, Rates, Inflation, Earnings, Yield.If a political headline contains high-weight financial keywords and a positive sentiment score, the dashboard flags it as a "Potential Long (Buy)" opportunity.⚠️ DisclaimerNote: This tool is for informational purposes only. Sentiment analysis is performed by a library (VADER) and may not capture the full complexity of global markets.
